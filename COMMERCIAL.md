# Коммерческое предложение
## Интеллектуальный ИИ-агент на базе DeepSeek (бесплатно) + Hermes Agent

### 1. Краткое описание продукта

Мы предлагаем готовое решение для бизнеса — **автономного ИИ-агента**, который работает 24/7, не требует оплаты за API-токены (использует бесплатный DeepSeek через собственный защищённый мост) и способен выполнять сложные задачи: от анализа данных до автоматизации бизнес-процессов.

Продукт построен на двух китах:
- **DeepSeek Provider Bridge** — бесплатный доступ к DeepSeek (Instant, Expert, Vision) через веб-интерфейс с PoW-солвером, поддержкой файлов и историей диалогов.
- **Hermes Agent** — самосовершенствующийся ИИ-агент с инструментами (терминал, файлы, браузер, cron, память, навыки), который использует DeepSeek как вычислительное ядро.

### 2. Ключевые возможности

#### 2.1. Автономное выполнение задач
- Самостоятельно планирует и выполняет многошаговые задачи (до 90 шагов).
- Использует инструменты: терминал, файловую систему, браузер, API, базы данных.
- Примеры: «Напиши парсер для этого сайта, запусти его раз в час, сохраняй результаты в Google Sheets и присылай мне отчёт в Telegram».

#### 2.2. Навыки (Skills) — библиотека готовых решений
- Создаёт и сохраняет навыки для повторяющихся задач.
- Навыки самосовершенствуются в процессе использования.
- Примеры навыков: «Деплой на сервер», «Мониторинг конкурентов», «Генерация отчётов», «Автоматическое тестирование».

#### 2.3. Память и обучение
- Запоминает предпочтения, структуру проектов, повторяющиеся действия.
- Строит диалектическую модель пользователя (предпочтения, стиль, контекст).
- Использует FTS5-поиск для извлечения информации из прошлых сессий.

#### 2.4. Планирование (Cron-джобы)
- Создаёт задания по расписанию с доставкой результатов в Telegram, Slack, Discord, Email.
- Пример: «Каждое утро в 9:00 проверяй доступность сайта, если упал — снимай скриншот и присылай в чат».

#### 2.5. Бесплатный DeepSeek через защищённый мост
- Безлимитные запросы к DeepSeek (Instant/Expert/Vision) за 0 рублей.
- Поддержка файлов (PDF, DOCX, TXT, CSV, ZIP, изображения, код, логи — 500+ форматов).
- Полная история диалогов в PostgreSQL с суммаризацией.

#### 2.6. Интеграции
- Telegram, Discord, Slack, WhatsApp, Signal, CLI.
- Базы данных (PostgreSQL, MySQL, MongoDB).
- Облачные провайдеры (AWS, GCP, Vercel).
- CRM (AmoCRM, Bitrix24, Salesforce).
- Google Sheets / Excel.

#### 2.7. Безопасность и изоляция
- Подтверждение критических команд (запрос разрешения).
- Изоляция через Docker, SSH, Modal (бессерверное состояние).
- Белый-лейбл брендинг (продажа под своим брендом).

### 4. Сценарии использования и примеры

#### 4.1. Для DevOps-команд
- **Автоматический деплой**: агент пушит код, запускает тесты, обновляет контейнеры.
- **Мониторинг**: проверяет доступность сервисов, логирует ошибки, отправляет уведомления.
- **Бэкапы**: создаёт и проверяет бэкапы баз данных по расписанию.

#### 4.2. Для аналитиков и маркетологов
- **Парсинг конкурентов**: агент собирает данные с сайтов, формирует отчёты.
- **Ежедневные дашборды**: собирает метрики из рекламных кабинетов, CRM и Google Analytics.
- **Анализ рынка**: ищет тренды, новости, упоминания бренда.

#### 4.3. Для бизнес-пользователей
- **Интеграция с CRM**: автоматическое обновление данных, создание задач, рассылка писем.
- **Генерация документов**: создаёт коммерческие предложения, акты, счета.
- **Обработка заявок**: анализирует входящие сообщения, классифицирует, назначает ответственных.

#### 4.4. Примеры сложных задач (выполняются агентом)
> «Проанализируй этот репозиторий, напиши документацию, создай навык для деплоя и запланируй еженедельный аудит кода».

> «Каждый час проверяй курс доллара на пяти сайтах, если среднее значение отличается от текущего на 2% — отправь уведомление в Telegram».
